# Demo Script Curriculum Manager (3 minuti)

## Scenario
Presentazione a stakeholders scuola italiana - manager curricolare read-only

## Passi Demo

1. **Apertura (30s)**
   - Aprire `index.html` nel browser
   - Mostrare header: "Manager Curricolo d'Istituto - Modelli sorgente"

2. **Navigazione (60s)**
   - Click "Modelli sorgente" → 10 template con badge "NON UFFICIALE"
   - Click "Documenti istituzionali" → catalogo read-only collegato
   - Click "Matrice revisione" → 10 documenti con checklist di revisione

3. **Affordance print (30s)**
   - Mostrare pulsante "Stampa / Salva in PDF"
   - Click → dialog browser → "Salva come PDF"

## Disclaimer
- App read-only, nessun documento ufficiale generato
- Richiede validazione umana prima di uso
- Export DOCX in roadmap futura
