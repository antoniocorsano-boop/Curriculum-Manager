# Istruzioni operative export istituzionali

## Per dirigente scolastico

1. Aprire `APRI_MANAGER_CURRICOLO_ISTITUTO.html`.
2. Entrare in **Documenti istituzionali**.
3. Usare preferibilmente i modelli **Report complessivo del curricolo d'istituto** e **Documento approvato / validato**.
4. Controllare titolo, anno scolastico, stato, versione, tabelle, decisioni e firme.
5. Usare l'export PDF/stampa per lettura o condivisione.
6. Usare Word `.doc` o DOCX quando il documento deve essere revisionato.

## Per referente curricolo

1. Verificare prima fonti, documenti e proposte nel Manager.
2. Entrare in **Documenti istituzionali**.
3. Esportare report complessivo, documento finale di dipartimento o documento approvato secondo necessita.
4. Conservare una copia di backup JSON/sessione prima di distribuire il lavoro.
5. Segnalare al dirigente quali documenti sono bozza, in revisione, validati o approvati.

## Per gruppo o dipartimento

1. Lavorare nel proprio ambito.
2. Compilare proposte, documenti e decisioni.
3. Usare **Documento prodotto dal gruppo di lavoro** per la consegna.
4. Usare **Documento finale di dipartimento** quando il dipartimento ha consolidato l'esito.
5. Inviare al referente il file modificabile e, se richiesto, il PDF.

## Esportare un documento approvato

1. Aprire **Documenti istituzionali**.
2. Selezionare **Documento approvato / validato**.
3. Esportare in DOCX o Word `.doc`.
4. Controllare che siano presenti estremi, fonti, documenti validati, stato e firme.
5. Procedere con approvazione, protocollo o conservazione solo fuori dal Manager, secondo le procedure ufficiali.

## Esportare un documento di gruppo

1. Aprire **Documenti istituzionali**.
2. Selezionare **Documento prodotto dal gruppo di lavoro**.
3. Esportare in Word `.doc` o DOCX per revisione.
4. Esportare in PDF/stampa per lettura e condivisione.
5. Verificare che non compaiano campi vuoti non spiegati o token tecnici.

## Salvare PDF

1. Cliccare **Stampa/PDF**.
2. Nel browser scegliere **Stampa**.
3. Selezionare **Salva come PDF**.
4. Controllare anteprima, pagine, tabelle e firme.
5. Salvare con nome chiaro e data.

## Aprire Word .doc

1. Cliccare **Word .doc**.
2. Aprire il file con Microsoft Word o LibreOffice.
3. Se il programma avvisa che il file proviene da HTML, confermare solo se il file e stato generato dal Manager.
4. Salvare eventualmente come `.docx` dopo revisione.

## Aprire DOCX

1. Cliccare **DOCX**.
2. Aprire il file con Microsoft Word o LibreOffice.
3. Controllare titolo reale del modello, tabelle, intestazione, sezioni, stato, versione, data e firme.
4. Se il file si apre correttamente, il gate di compatibilita applicativa e superato per quell'ambiente.

## Backup

1. Usare **Esporta backup** per salvare i dati JSON.
2. Usare **Esporta sessione** per condividere o ripristinare una sessione di lavoro.
3. Conservare il backup insieme alla data e al nome del gruppo o referente.

## Ripristinare una sessione

1. Aprire il Manager sul PC di destinazione.
2. Usare la funzione di import sessione o backup disponibile nel Manager.
3. Verificare dashboard, documenti e proposte dopo il ripristino.
4. Eseguire un nuovo export di prova.

## Se Word o LibreOffice segnala problemi

1. Provare prima il formato alternativo: Word `.doc` se il DOCX non si apre, oppure DOCX se il `.doc` viene interpretato male.
2. Verificare che il file non sia stato scaricato parzialmente.
3. Rigenerare il file dal Manager.
4. Aprire il file su un secondo PC o con l'altro programma.
5. Se il problema persiste, usare temporaneamente PDF/stampa e segnalare il caso come gate di compatibilita applicativa.
