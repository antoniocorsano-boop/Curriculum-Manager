# Template documenti istituzionali curricolari

Questi file sono **template sorgente** per documenti istituzionali curricolari del Manager Curricolo d’Istituto.

## Avvertenze

- Non sono documenti ufficiali.
- Non contengono dati reali.
- Non sostituiscono delibere, atti o procedure dell’Istituto.
- Richiedono compilazione umana, verifica normativa e validazione da parte dei soggetti competenti.
- L’export DOCX/PDF è fuori scope di questa raccolta: i file sono sorgenti Markdown compilabili.
- La verifica visuale in LibreOffice Writer o Microsoft Word resta gate esterno.
