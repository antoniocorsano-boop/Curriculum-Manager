## Decisioni prese

1. **Formula obbligatoria adottata**: user need → fase → documento/attività → azione umana → output → stato finale

2. **Pattern visuale vincolato**: ogni vista deve mostrare dove si è, cosa si fa, perché, cosa si può fare, cosa si produrrà

3. **Stati del lavoro definiti**: Da avviare → In corso → Annotato → Pronto per confronto → Consolidato → Da rivedere → Incluso nel pacchetto → Esportato/stampato

4. **Profilo non è account**: serve solo a filtrare finché non esiste autenticazione reale

5. **Sincronizzazione futura**: V1 locale, V2 con OAuth/Drive (nessun gate MGR-048)

6. **Formula UX obbligatoria**: dove sono → cosa faccio → perché → cosa posso fare → cosa produco
