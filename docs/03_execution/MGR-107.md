## Percorso demo passo-passo

### 1. Apertura app
- Apri `http://localhost:5173` via HTTP (non file://)
- Verifica header "Manager Curricolo d'Istituto"
- Verifica sidebar con 7 voci

### 2. Navigazione principale
- Clicca ogni voce sidebar:
  - Materiali
  - Documenti
  - Output
  - Revisione
  - Mappa
  - Stati attività
  - Wiki
- Verifica che ogni vista si apra senza errori

### 3. Matrice Revisione
- Apri "Revisione"
- Verifica summary grid con 5 contatori
- Verifica card documenti con textarea bozza
- Pulsante "Reset annotazioni" funziona con conferma

### 4. Output Center
- Apri "Output"
- Verifica summary grid con 5 contatori
- Verifica toolbar con Esporta JSON, Esporta Markdown, Reset tutti
- Verifica card con stato "Bozza non modificata"

### 5. Draft locale
- Apri "Documenti"
- Clicca su un documento
- Verifica textarea modificabile
- Digita contenuto
- Clicca "Salva bozza"
- Verifica stato "Salvato su questo dispositivo"
- Clicca "Stampa bozza"

### 6. Export locale
- Clicca "Esporta JSON" - verifica download
- Clicca "Esporta Markdown" - verifica download

### 7. Reset Output Center (opzionale)
- Clicca "Reset tutti" SOLO se necessario
- Conferma esplicita richiesta

## Stop rule

La demo si ferma se:

- Una vista non si apre
- Un pulsante non produce effetto
- Compare riferimento a rete/cloud/backend
- Export/reset tocca dati fuori perimetro
- Local-only non è chiaro
- Dati reali sono presenti
- Reset non è comprensibile
