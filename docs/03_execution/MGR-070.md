# MGR-070 — PRINTABLE_DRAFT_GATE

## Obiettivo

Aggiungere una modalità stampa/anteprima stampabile della bozza locale del documento, senza introdurre export DOCX/PDF programmatico.

## File modificati

- `src/views/documentiIstituzionaliView.js` (pulsante "Stampa bozza", div contenuto stampabile)
- `src/styles.css` (CSS print per bozza documento)

## Implementazione

### CTA stampa
```html
<button type="button" id="printDraftButton" class="action secondary">Stampa bozza</button>
```

### JS stampa
```javascript
const printDraftButton = document.getElementById("printDraftButton");
if (printDraftButton) printDraftButton.addEventListener("click", () => window.print());
```

### Print styles
- `.no-print` nascosto (sidebar, toolbar, pulsanti)
- `.editable-draft-textarea` nascosto in stampa
- `.print-draft-content` mostrato in stampa con stile leggibile
- Titolo, stato e sezioni visibili
