# MGR-073 - Document Output Center implementation

## Obiettivo

Implementare un Centro output documenti minimo, locale e navigabile, coerente con `DOCUMENT_OUTPUT_CENTER_CONTRACT`.

## Implementazione

- Nuova vista `documentOutputCenter`.
- Accesso dalla sidebar con voce `Output`.
- Accesso dalla vista Documenti tramite pulsante `Apri Centro output documenti`.
- Stato locale separato in `localStorage` con chiave `curriculumManager.documentOutputState`.
- Riepilogo aggregato:
  - documenti iniziati;
  - bozze salvate;
  - pronti per stampa;
  - da rivedere;
  - completati manualmente.
- Azioni per documento:
  - apri documento;
  - stampa bozza con `window.print()`;
  - copia riepilogo negli appunti;
  - segna da rivedere;
  - completato manualmente;
  - reset locale del singolo documento.

## Confini rispettati

- Nessun backend.
- Nessuna API.
- Nessun OAuth.
- Nessuna sincronizzazione remota.
- Nessun export DOCX.
- Nessun PDF programmatico.
- Stampa solo tramite browser.
- Stati locali separati da validazione, approvazione, protocollo o invio.

## Marker

- `OUTPUT_CENTER_IS_LOCAL_STATUS_ONLY`
