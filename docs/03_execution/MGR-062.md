## Comportamento di instradamento

### getSuggestedPath(roleId, profile)
- Se ruolo assente: mostra "Orientamento" con invito a completare profilo
- Se ruolo presente: mostra percorso suggerito dal ROLE_WORK_PATHS_CATALOG

### renderSuggestedContext(profile)
- Mostra contesto: Ambito, Scuola, Ordine (se presenti)
- Non filtrano né bloccano accessi

### openHomeGuidedPath()
- Naviga alla vista suggerita dal ruolo (actions[0] del ROLE_WORK_PATHS_CATALOG)

## Esempio instradamento per ruolo

| Ruolo | Azione suggerita | Descrizione |
|-------|------------------|-------------|
| docente | documentiIstituzionali | Consultare modelli sorgente, verificare template disciplina |
| coordinatore | matriceRevisione | Gestire documento dipartimento, verificare coerenza |
| gruppo | matriceRevisione | Aprire documento gruppo lavoro |
| staff | modelliSorgente | Archiviare revisioni, supportare esportazione |
