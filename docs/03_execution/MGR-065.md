## Rischi residui

1. **Contenuti duplicati**: Alcuni documenti potrebbero avere sezioni sovrapposte (es. programmazione + curricolo ordine)
2. **Priorità multiple**: 3 programmazioni annuali mappate allo stesso target
3. **Sezioni non allineate**: I mapping section sono indicativi e potrebbero richiedere aggiustamenti
4. **File protocol**: Nessun fetch runtime implementato, i path sono riferimenti solo
